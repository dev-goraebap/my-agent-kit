# Lean PRD — 주문 서비스

## Overview
사용자가 장바구니 상품을 결제 완료까지 진행할 수 있는 최소 주문 흐름.

## Background
현재 상품 상세 페이지는 있으나 구매 완료 경로가 없어 사용자가 이탈한다. 경쟁사 대비 전환율이 현저히 낮고, 고객센터 문의의 30%가 "어떻게 사는지 모르겠다"이다.

## Goals
- 2026-Q2 GA 출시
- 장바구니 진입 → 결제 완료 평균 소요 시간 90초 이하
- 주문 실패율 2% 이하 (결제 승인 실패 + 시스템 오류 합산)
- 재고·결제 상태 불일치 건수 0건 유지
- 런칭 후 30일 내 일일 주문 1,000건 달성

## Non-goals
- 쿠폰·프로모션 시스템 (별도 팀 소관)
- 정기 결제·구독
- 다중 통화 / 해외 배송
