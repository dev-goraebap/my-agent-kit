Components
Components are interactive building blocks for creating a user interface. They can be organized into categories based on their purpose: Action, containment, communication, navigation, selection, and text input.
Buttons

Button groups
Button groups organize buttons and add interactions between them
Buttons
Buttons prompt most actions in a UI
Extended FABs
Extended floating action buttons (extended FABs) help people take primary actions
FAB menu
The floating action button (FAB) menu opens from a FAB to display multiple related actions
Floating action buttons (FABs)
Floating action buttons (FABs) help people take primary actions
Icon buttons
Icon buttons help people take actions with a single tap
Segmented buttons
Segmented buttons help people select options, switch views, or sort elements
Split buttons
Split buttons open a menu to give people more options related to an action
Date & time pickers

Date pickers
Date pickers let people select a date, or a range of dates
Time pickers
Time pickers help people select and set a specific time
Loading & progress

Loading indicator
Loading indicators show the progress of a process for a short wait time
Progress indicators
Progress indicators show the status of a process in real time
Navigation

Navigation bar
Navigation bars let people switch between UI views on smaller devices
Navigation drawer
Navigation drawers let people switch between UI views on larger devices
Navigation rail
Navigation rails let people switch between UI views on mid-sized devices
Sheets

Bottom sheets
Bottom sheets show secondary content anchored to the bottom of the screen
Side sheets
Side sheets show secondary content anchored to the side of the screen
All other components

App bars
App bars are placed at the top of the screen to help people navigate through a product
Badges
Badges show notifications, counts, or status information on navigation items and icons
Cards
Cards display content and actions about a single subject
Carousel
Carousels show a collection of items that can be scrolled on and off the screen
Checkbox
Checkboxes let users select one or more items from a list, or turn an item on or off
Chips
Chips help people enter information, make selections, filter content, or trigger actions
Dialogs
Dialogs provide important prompts in a user flow
Divider
Dividers are thin lines that group content in lists or other containers
Lists
Lists are continuous, vertical indexes of text and images
Menus
Menus display a list of choices on a temporary surface
Radio button
Radio buttons let people select one option from a set of options
Search
Search lets people enter a keyword or phrase to get relevant information
Sliders
Sliders allow users to make selections from a range of values
Snackbar
Snackbars show short updates about app processes at the bottom of the screen
Switch
Switches toggle the selection of an item on and off
Tabs
Tabs organize content across different screens and views
Text fields
Text fields let users enter text into a UI
Toolbars
Toolbars display frequently used actions relevant to the current page
Tooltips
Tooltips display brief labels or messages
