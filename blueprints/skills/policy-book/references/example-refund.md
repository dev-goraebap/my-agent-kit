# POL-0001 환불 정책

## Context
2025년 4분기 CS 리포트에서 환불 관련 문의가 전체의 32%(월평균 240건)를 차지했고, 그중 절반 이상이 "환불 가능 기간이 불명확하다"는 내용이었다. 동시에 전자상거래법 제17조 준수 기준을 통일할 필요가 있어 공식 정책을 수립한다.

## Decision
- **일반 회원**: 주문 후 7일 이내 환불 허용
- **VIP 회원**: 주문 후 14일 이내 환불 허용
- 공통 조건: 상품 미개봉, 주문 상태가 `PAID`
- 환불은 결제 수단으로 원복 (부분 환불은 지원하지 않음)

## Consequences
- **긍정**: 환불 관련 CS 문의 감소 예상 (월 240건 → 150건 이하 목표)
- **긍정**: VIP 혜택 차별화로 등급 상승 동기 부여
- **부정**: 재고 관리 복잡도 증가 (반품 상품 검수·재입고 프로세스 필요)
- **부정**: 기존 임의 처리되던 케이스가 일괄 거부될 수 있어 초기 반발 우려

## Exceptions
- **식품·화장품**: 개봉 시 환불 불가 (식품위생법·화장품법 준수)
- **해외 배송**: 환불 기간 계산 시 배송 소요일 제외
- **이벤트 상품**: 프로모션 종료 후 환불 불가 (개별 고지)

## 처리 절차
1. 사용자가 환불 요청 접수
2. 주문 상태 `PAID` 여부 + 기간 조건 검증
3. 상품 미개봉 확인 (CS 판단 또는 사진 검수)
4. 결제 취소 호출 (Payment 도메인)
5. 재고 복구 (Inventory 도메인)
6. `OrderCanceled` 이벤트 발행
7. 사용자에게 환불 완료 알림 (Notification 도메인)

## 근거
- [전자상거래법 제17조 — 청약철회](https://www.law.go.kr/법령/전자상거래등에서의소비자보호에관한법률/제17조) — 7일 법정 최소 기간 준수
- 2025 Q4 CS 리포트 — 내부 문서
- 2025-12-15 기획 회의 결정 (VIP 14일 확대)
